Project DB Arkanoid

Goku and Bulma go on an adventure
to find the seven mystical
Dragon Balls (ドラゴンボール),
which have the ability to summon the powerful dragon Shenron, who can grant whoever summoned him their greatest desire.
Help Goku in this amazing journey!

GAME INSTRUCTIONS

  • Use the "s" button on your keyboard to start the game
  • Once in the game, you can move the paddle using the "arrowLeft" and the "arrowRight" keys
  • Use the "Esc" key yo pause the game in any moment
  • May they be some hidden keys?
